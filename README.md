# Jarvis Engineer (Python Edition)

Este projeto é um agente de IA simples que automatiza a criação de componentes em React Native.

## Requisitos

- Python 3.8+
- Git configurado
- Node.js (para rodar os testes)
- OpenAI API Key (opcional, para gerar código real)
